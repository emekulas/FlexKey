=== FlexKey RFID Access Control System ===

RFID-based triggering system for smart home automation. Offers device control and/or web service calls when card is read. Group-based authorization enables different automation scenarios per card. Remote management via web interface with customizable triggering levels.

=== Use Cases ===
- Home automation triggering
- Device control + notification
- Automation scenarios
- Remote system management
- Smart home integration
- Multi-device control
